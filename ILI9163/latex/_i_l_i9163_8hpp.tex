ILI9163 LCD color display. 

This class implements an interface to an 130 x 130 pixel color LCD display. The interface is SPI 4 pins (SCL, SDA, CS, A0). The driver chip is an ILI9163 a-\/\+Si TFT LCD .

The interface is direct\+: all writes will be directly written to the display.
